# Cortex - Project Rules

1. **Tech Stack:** Stick strictly to Tauri, React, TypeScript, and TailwindCSS.
2. **Local-First:** Do not add any cloud telemetry or external API calls for memory storage. All data must be read from and written to the local PC.
3. **Aesthetics:** The UI must feel premium and native. Use dark mode by default with subtle, glowing accents.
4. **Code Quality:** Use standard TypeScript best practices. Keep components small and focused.
